Now that the repo is fully set up, let's configure Claude for this course.

## Choosing Your Model

The first question you'll likely have is which model to use. You can select your model by specifying `/model` and running it to set the model for Claude Code.

The best approach is to go with the defaults, but here's the tricky part: your defaults will differ based on your subscription tier.

| Subscription Tier | Default Model |
|---|---|
| 5X Max | Opus 4.6 |
| Pro | Sonnet 4.6 |

**Do not use Haiku** at any point during this course. While it's fast, it simply isn't capable enough to handle the workflows you'll encounter.

### Setting the Model

To select your model, you can press enter here and you'll see confirmation:

![Terminal showing 'set model to default Opus 4.6' confirmation](https://res.cloudinary.com/total-typescript/image/upload/v1773410399/ai-hero-images/osdk1ww4atapjwuurgmn.png)

### Effort Level

I also recommend choosing medium effort. You can adjust this left and right for low or high effort options.

![Claude Code interface showing effort level selection with low, medium, and high options](https://res.cloudinary.com/total-typescript/image/upload/v1773410400/ai-hero-images/r4pxiascnnf8uvx6sfn0.png)

I suggest whatever the defaults suggest. Anthropic has done internal evaluations showing that medium effort works fine for most work.

## Choosing Your Subscription

Your next question will probably be about which subscription to get. You have three main options:

- Pro
- 5X Max
- 20X Max

![Claude subscription tier options showing Pro, Max 5X, and Max 20X plans](https://res.cloudinary.com/total-typescript/image/upload/v1773410401/ai-hero-images/dxzrjvpjchliokypizdm.png)

I can't give a universal recommendation because I don't know your budget constraints or how much you'll use Claude going forward. You can [compare plans and pricing](https://support.claude.com/en/articles/11049762-choosing-a-claude-plan) on the Anthropic site.

### My Recommendation

I created this course with **5X Max**. If you want to use the best model Claude has available (Opus 4.6), I recommend getting 5X Max, even if it's just for a month.

However, if you can't stretch that far or want to try it out first, **Pro with Sonnet 4.6** will work. You should be able to get through the course with Pro, though I can't guarantee you won't hit usage limits if you're not diving deep into every lesson.

### Summary

- Use the default model for your subscription tier
- Choose between Pro or 5X Max based on your budget
