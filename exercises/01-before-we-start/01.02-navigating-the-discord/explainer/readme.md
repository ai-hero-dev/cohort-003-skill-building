One of the most important places you're going to go during this cohort is the [Discord](https://aihero.dev/discord).

There are a few places you need to know about. I use this Discord for cohorts, but also for general community chat. So there are channels like `#general`, `#evalite`, `#ai-hero` that anyone can access.

### Your Cohort Role

As a cohort member, you should have the `Cohort 003` role. If you don't have this role, ping me via DM so you can get it.

Once you have that role, you'll see the `#cohort-003` category, and you'll be able to see three channels inside it.

| Channel                 | Purpose                                                                                                              |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------- |
| `#cohort-003-general`   | General cohort announcements and general chats                                                                       |
| `#cohort-003-random`    | Any random stuff you want to post. Why not post a picture of your pets in there?                                     |
| `#cohort-003-questions` | Asking questions about the course. Any discussions in this channel will go inside the cohort three questions thread. |

![Cohort 3 category expanded showing three channels](https://res.cloudinary.com/total-typescript/image/upload/v1773411167/ai-hero-images/wp7p9n5urqmjtamidonc.png)

## How to Choose the Right Channel

When you post something, here's your thought process:

| Question                                                           | Answer | Channel                 |
| ------------------------------------------------------------------ | ------ | ----------------------- |
| Is it totally random, not related to the course at all?            | Yes    | `#cohort-003-random`    |
| Is it a question about the course material? Do you want an answer? | Yes    | `#cohort-003-questions` |
| Is it related to the course but not a specific question?           | Yes    | `#cohort-003-general`   |

Before posting a question, you can search to see if someone has asked it before and if there are any answers already available.
